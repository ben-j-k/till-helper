# Till Helper
Till helper is a WinForms application used to assist retail managers during cash-up.

## Features
- Reduce human error by calculating cash takes based on quantity of each type of denomination in the till
- Report variance between actual and expected takes
- Support for petty cash reporting
- Support for multiple tills
- Support for cash and card payments
- Generate a report that includes the manager responsible, along with additional comments that can be exported to CSV and stored for reference
